1、打开控制面板（`shift + chommand + p`）；

2、输入 `shell command`;

在提示里面看到 `Shell Command: Install 'code' command in Path`, 按回车即可

3、在终端中使用命令行;

```shell
 # 打开vscode
 code

 # 打开当前文件夹
 code ./

```
